# Основная часть по созданию и настройке Git

Публичный репозиторий проекта:  
https://github.com/DmitryLLN/deloproizvodstvo

## 1. Проверка установленной версии Git
В данном подразделе должны быть приведены команда проверки версии Git и screenshot результата.

## 2. Настройка Git
В данном подразделе должны быть приведены команды git config и screenshot результата.

## 3. Проверка состояния файлов
В данном подразделе должны быть приведены команды git status до и после индексации файлов.

## 4. Добавление файлов и создание коммита
В данном подразделе должны быть приведены команды git add и git commit с подтверждающими screenshot.

## 5. Работа с ветками и stash
В данном подразделе должны быть приведены команды создания ветки, просмотра веток и использования git stash.

## 6. Работа с внешним репозиторием
В данном подразделе должны быть приведены команды git remote, git push, git clone, git fetch и git pull.

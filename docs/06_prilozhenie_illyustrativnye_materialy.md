# Приложение. Перечень иллюстративных материалов

В приложение включаются screenshot, подтверждающие выполнение основных этапов настройки Git, ведения локального репозитория и публикации проекта в удаленном публичном репозитории GitHub.

## Состав приложения

1. Screenshot 1 — Проверка установленной версии Git.
2. Screenshot 2 — Глобальная конфигурация Git.
3. Screenshot 3 — Проверка состояния локального репозитория.
4. Screenshot 4 — Структура каталогов проекта на локальном компьютере.
5. Screenshot 5 — Проверка связи с удаленным репозиторием.
6. Screenshot 6 — Просмотр списка веток.
7. Screenshot 7 — Просмотр истории коммитов.
8. Screenshot 8 — Создание новой ветки.
9. Screenshot 9 — Состояние файлов до индексации.
10. Screenshot 10 — Состояние файлов после индексации.
11. Screenshot 11 — Создание коммита.
12. Screenshot 12 — Использование команды git stash.
13. Screenshot 13 — Выполнение команды git fetch.
14. Screenshot 14 — Выполнение команды git pull.
15. Screenshot 15 — Клонирование удаленного репозитория.
16. Screenshot 16 — Главная страница публичного репозитория GitHub.
17. Screenshot 17 — Структура каталогов проекта в GitHub.
18. Screenshot 18 — Отображение README проекта на GitHub.

## Назначение приложения

Приложение предназначено для подтверждения практического выполнения всех основных этапов рейтинговой работы и обеспечивает наглядность представления результатов.

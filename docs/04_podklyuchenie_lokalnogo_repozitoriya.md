# Настройка и подключение локального репозитория к проекту

## Алгоритм подключения

1. Создание публичного репозитория на GitHub.
2. Клонирование удаленного репозитория на локальный компьютер.
3. Проверка связи локального репозитория с удаленным.
4. Создание и изменение файлов проекта.
5. Индексация изменений.
6. Создание коммита.
7. Отправка изменений в удаленный репозиторий.
8. Проверка отображения изменений на сайте GitHub.

Каждый шаг должен быть подтвержден соответствующим screenshot.

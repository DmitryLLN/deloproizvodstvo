# Иерархическая структура работ проекта

## Выбранная модель жизненного цикла
Для проекта информационной системы «Делопроизводство» выбрана каскадная модель жизненного цикла.

## ИСР проекта

1. Инициация проекта
   1.1. Формулирование цели проекта
   1.2. Определение объекта автоматизации
   1.3. Определение границ проекта

2. Анализ предметной области
   2.1. Исследование процессов делопроизводства
   2.2. Выявление участников документооборота
   2.3. Выделение видов документов

3. Формирование требований
   3.1. Функциональные требования
   3.2. Нефункциональные требования
   3.3. Требования к данным

4. Проектирование системы
   4.1. Проектирование структуры данных
   4.2. Проектирование ролей пользователей
   4.3. Проектирование интерфейсов

5. Организация проектного репозитория
   5.1. Настройка Git
   5.2. Создание локального репозитория
   5.3. Подключение удаленного репозитория

6. Завершение проекта
   6.1. Подготовка итоговой документации
   6.2. Подготовка screenshot
   6.3. Сдача работы

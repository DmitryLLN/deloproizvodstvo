# Заключение

В ходе выполнения рейтинговой работы были определены основания для разработки информационной системы «Делопроизводство» и установлено ее назначение как средства автоматизации процессов регистрации, учета, хранения, маршрутизации, поиска и контроля исполнения документов в организации.

В соответствии с моделью жизненного цикла информационной системы была построена иерархическая структура работ проекта, позволившая декомпозировать проект на логически взаимосвязанные этапы: инициацию, анализ предметной области, формирование требований, проектирование, организацию проектного репозитория и завершение проекта.

В практической части работы была выполнена настройка и использование системы управления версиями Git. Были освоены основные операции: проверка конфигурации, контроль состояния файлов, индексирование изменений, создание коммитов, работа с ветками, временное сохранение незавершенных изменений и просмотр истории проекта.

Кроме того, был создан публичный удаленный репозиторий на платформе GitHub и выполнено подключение локального репозитория к удаленному репозиторию проекта. Это обеспечило возможность публикации материалов проекта в сети Интернет, контроля версий и хранения истории изменений.

Таким образом, поставленная цель работы достигнута: проект информационной системы «Делопроизводство» был организован как учебный ИТ-проект, сопровождаемый средствами Git и GitHub, а все основные этапы, предусмотренные заданием, были выполнены.

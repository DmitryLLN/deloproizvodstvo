# Анализ предметной области

Информационная система «Делопроизводство» предназначена для автоматизации следующих процессов:

- регистрация входящих документов;
- регистрация исходящих документов;
- регистрация внутренних документов;
- назначение исполнителей;
- контроль сроков исполнения;
- поиск документов по реквизитам;
- хранение электронных копий документов.

Основные виды документов:
- входящий документ;
- исходящий документ;
- внутренний документ;
- служебная записка;
- приказ;
- распоряжение.

# Категории пользователей и права доступа

| Категория пользователя | Просмотр | Создание | Редактирование | Согласование | Назначение исполнителей | Администрирование |
|---|---|---|---|---|---|---|
| Администратор | Да | Да | Да | Да | Да | Да |
| Руководитель | Да | Нет | Ограниченно | Да | Да | Нет |
| Делопроизводитель | Да | Да | Да | Нет | Нет | Нет |
| Исполнитель | Да | Нет | Ограниченно | Нет | Нет | Нет |

## Пояснение
Администратор обладает полными правами.
Руководитель утверждает документы и назначает исполнителей.
Делопроизводитель регистрирует документы и ведет их учет.
Исполнитель работает только с назначенными ему документами.

# Модель жизненного цикла проекта ИС

Для проекта информационной системы «Делопроизводство» выбрана каскадная модель жизненного цикла.

Этапы:
1. Инициация.
2. Анализ предметной области.
3. Формирование требований.
4. Проектирование.
5. Организация репозитория и сопровождение документации.
6. Завершение проекта.

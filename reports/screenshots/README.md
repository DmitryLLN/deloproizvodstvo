# Папка screenshots

В данной папке размещаются графические файлы screenshot, подтверждающие выполнение основных этапов рейтинговой работы.

Рекомендуется сохранять screenshot в формате PNG с последовательной нумерацией.

Пример имен файлов:
- screenshot_01_git_version.png
- screenshot_02_git_config.png
- screenshot_03_git_status.png
- screenshot_04_project_tree.png

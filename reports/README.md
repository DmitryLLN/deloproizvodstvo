# Отчетные материалы проекта

В данной папке размещаются материалы, подтверждающие выполнение рейтинговой работы по проекту информационной системы «Делопроизводство».

## Состав папки

- README.md — общее описание отчетных материалов;
- 01_perechen_skrinshotov.md — перечень screenshot, используемых в работе;
- 02_rekomendacii_po_oformleniyu.md — рекомендации по включению screenshot в итоговый документ;
- screenshots — папка для графических файлов screenshot.

## Назначение screenshot

Screenshot используются для подтверждения:
- установки и настройки Git;
- состояния локального репозитория;
- выполнения операций add, commit, push;
- работы с ветками;
- использования stash;
- работы с удаленным репозиторием GitHub;
- публикации материалов проекта в публичном репозитории.

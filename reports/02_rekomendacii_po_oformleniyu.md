## Рекомендации по включению в Word

1. Каждый screenshot следует снабжать подписью.
2. Подпись должна содержать номер и краткое наименование изображения.
3. Screenshot рекомендуется размещать либо по ходу текста, либо в приложении.
4. Для команд Git лучше сохранять screenshot так, чтобы была видна и сама команда, и результат ее выполнения.
5. Для screenshot с GitHub желательно, чтобы на изображении были видны имя репозитория и структура проекта.
